Changelog

Version history and patch notes for Dark Pawns.

v2.1.0 — 2026-04-28 — Website Launch

  • Hugo static site deployed with cream paper + oxblood design system
  • 475 help files published (commands, info, socials, spells, wizhelp)
  • Class, race, and skill descriptions from original website
  • World lore (Friar Drake letter, creation myth) archived
  • Historical player quotes and community content preserved
  • Web client with cream-chrome wrapper and dark terminal
  • AI-readable: llms.txt, Markdown content negotiation, JSON-LD structured data
  • Zero JavaScript in site templates — CSS-only mobile nav
  • Responsive, accessible (WCAG AA), print stylesheet

v2.0.0 — 2025-06-01 — The Go Port

  • Complete CircleMUD C → Go server port
  • WebSocket protocol for browser-based play
  • AI agent protocol for autonomous player agents
  • All original areas, classes, races, skills preserved
  • Mobile AI system functional
  • Combat system, saving throws, spell system ported
  • Open-sourced on GitHub

v3.0 beta — 2003-2005 — Final Original (Historical)

  • Mobile AI with intelligent mob behavior
  • Magical tattoos and talking weaponry
  • New classes and areas
  • Clan system updates
  • (Preserved from original darkpawns.com)

v1.0 — 1997 — The Beginning (Historical)

  • CircleMUD-based dark fantasy MUD
  • Created by Serapis (Derek Karnes)
  • Original class system: Assassin, Magus, Monk, Ninja, Paladin, Ranger, Warrior
  • Race system with alignment-based restrictions
  • PK-enabled, permadeath for certain zones
