'CHARM PERSON'

April 28, 2026

Usage : cast ‘charm person’ Accumulative: No

It is not possible to charm anyone that is already charmed, if you’re charmed yourself, or if the person is higher level than yourself. The person will get a saving throw versus “paralysis”.

A charmed person will follow you, and not be able to attack you. You can order the charmed person using the order command. Monsters don’t like being charmed, and a failed attempt will usually result in your being attacked.

See also: FOLLOW, ORDER
