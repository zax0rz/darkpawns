USERS

April 28, 2026

Usage: users [switches]

USERS gives a list of all sockets (i.e., connections) currently active on the MUD. The multi-column display shows the socket number (used by DC), class, level, and name of the player connected, connection state, idle time, and hostname.

The following switches are available:

-k or -o Show only outlaws (killers and thieves). -p Show only sockets in the playing sockets. -d Show only non-playing (deadweight) sockets. -l min-max Show only sockets whose characters are from level min to max. -n Show the socket with associated with it. -h Show all sockets from . -c list Show only sockets whose characters’ classes are in list.

See also: DC, SLOWNS wizonly
