CHECKLOAD

April 28, 2026

Usage: checkload < obj | mob >

Used to check where a particular mobile or object loads given its virtual number.

Examples:

checkload o 1300 (where does object 1300 load?) checkload m 1315 (where does mobile 1315 load?)
