SET

April 28, 2026

Usage: set [ file | player ]

SET is an extremely powerful command, capable of setting dozens of aspects of characters, both players and mobiles.

SET PLAYER forces set to look for a player and not a mobile; useful for players with names such as ‘guard’.

SET FILE lets you change players who are not logged on. If you use SET FILE on a player who IS logged on, your change will be lost. If you wish to set a player who is in the game but is linkless, use set twice – once with the FILE argument, and once without – to make sure that the change takes.

For toggled fields (BINARY), the value must be ON, OFF, YES, or NO.
