Usage: check
       mail <person>
       receive
       
The commands used by the Dark Pawns Mail System (DPMS) are MAIL, CHECK, and
RECEIVE.  These commands can only be used while in an Official DPMS Post
Office.  You must be at least second level in order to use the DPMS.

Examples:

  To check and see if you have any mail waiting:
  > check

  To write mail to Niandra:
  > mail Niandra

  To ask the Postmaster to give you your mail, if you have any waiting:
  > receive

Stamps cost 25 coins.
