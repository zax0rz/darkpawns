Usage: donate <object>
       donate <number> coins


This command takes an object that you no longer want and deposits it in the
donation room of Kir Drax'in to assist newbies in getting started.  There is
a 50% chance objects donated will wind up in a donation room in Kir Drax'in,
a 25% chance objects donated will wind up in a donation room in Kir Oshi, and 
a 25% chance the item will be junked.

  > donate shield
  > donate all.bread
  > donate 500 coins

See also: JUNK
