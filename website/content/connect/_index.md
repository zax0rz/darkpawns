## Web Client

Play in your browser at [darkpawns.labz0rz.com/play](https://darkpawns.labz0rz.com/play). No downloads, no installation. Works in any modern browser. Suitable for casual play.

## Telnet

- **Address:** `darkpawns.labz0rz.com`
- **Port:** `4000`

```
telnet darkpawns.labz0rz.com 4000
```

## Recommended MUD Clients

For extended play sessions, a dedicated client provides scripting, auto-mapping, and scrollback that a raw terminal or browser can't match.

| Client | Platform | Notes | Link |
|--------|----------|-------|------|
| **Mudlet** | Windows, macOS, Linux | Free. Built-in scripting and auto-mapper. | [mudlet.org](https://www.mudlet.org/) |
| **TinTin++** | macOS, Linux | Terminal-based. Power user oriented. | [tintin.mudhalla.net](https://tintin.mudhalla.net/) |
| **TinyFugue** | macOS, Linux | UNIX classic. Lightweight. | `apt install tf` / `brew install tinyfugue` |

## Getting Started

1. Create a character. Names must be unique.
2. Choose your race and class.
3. Read the room descriptions. They contain information relevant to survival.
4. Type `help` for a command list. Type `help <command>` for usage details.
5. Do not junk stolen equipment. There is no recovery mechanism.
