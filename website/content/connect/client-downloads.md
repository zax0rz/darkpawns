## mud clients

   [gMUD v1.9b](http://www.dp-players.com/download/download.php?file=0) (206kb)
   -
   Your standard MUD client. Freeware, ANSI support, relatively small scrollback buffer,
   trigger and alias support.


   [MUSHClient v3.45](http://www.dp-players.com/download/download.php?file=1)
   (1.35mb) -
   One of the more feature-rich MUD clients available. Seemingly endless scrollback buffer;
   this one seems to be more popular with the trigger-reliant mudders. This is
   shareware, so the full program costs $20. Or you can use a [serial code](http://www.dp-players.com/go.php?dp=files/mushclient_serial.mud).


   [zMUD v6.16](http://www.dp-players.com/download/download.php?file=2)
   (4.79mb) -
   Probably one of the most popular clients available. This is a shareware version which
   means you can only use it for 30 days before you have to pay the $25 registration
   fee. Or you can use the provided [crack](http://www.dp-players.com/download/download.php?file=5)
   (188kb) - consult DAMN.NFO for instructions.


   [MCL v0.53.00](http://www.dp-players.com/download/download.php?file=6)
   (162kb) - MUD Client for Linux.
