# Dark Pawns — History

*Originally written by Orodreth circa 1999. Updated by Frontline circa 2004.*
*Source: dp-players.com (archived 2004-07-28 via web.archive.org)*

---

**September 1994** — DP is born. Based on CircleMUD 1.7, running on knight.ufp.org. Primary developers: Serapis (D. Karnes) and Tracer (C. Jackson).

**December 1995** — Base code upgraded to CircleMUD 2.2. This becomes known as DP v1.0.

**February 1996** — Arak (B. Parker) and Orodreth (S. Thompson) join as coders. Oddity (A. McConnell) joins as head builder.

**March 1996** — Base changed to CircleMUD 3.0pl9 (beta). Frontline (R. Paret) joins as coder. Nimos and Aralius (R. Martin) join as builders. Known as DP v2.0.

**June 1996** — Upgraded to CircleMUD 3.0pl11.

**December 1996** — Moved to augusta.net.

**May 1997** — Moves to pawns.guru.org (mystech.com, Serapis' employer). First public announcement on rec.games.mud.announce. Opening day: Wednesday, May 21, 1997.

**October 1997** — Moved to darkrune.guru.org, hosted at the University of Virginia by Orodreth. Still advertised as pawns.guru.org. Orodreth and Serapis become co-implementors. All inactive immortals fired/deleted. Oddity leaves.

**May 1998** — Hostname changed to ritual.darkrune.org (also pawns.darkrune.org). DP v2.1 unveiled. Serapis leaves. Frontline and Orodreth become co-implementors.

**December 1998** — Moves to alcatraz.wolfpaw.net (hosted free by wolfpaw.net in exchange for advertising). Port changes from 4000 to 4300 for the first time. Aliased as pawns.wolfpaw.net and pawns.darkrune.org. DP v2.2 released — largest update in DP's history. Delirium and Hulk join the immortal staff. Dark Pawns: Age of Cities (v3.0) announced.

**October 1999** — Staff shake-up. Delirium, Hulk, and all Chosen demoted/fired.

**December 1999** — Doragar joins as builder/coder.

**April 2000** — Free wolfpaw account expires. DP moves to talon.wolfpaw.net (now paying). Keeps pawns.wolfpaw.net and pawns.darkrune.org as aliases.

**February 2001** — Disaster. A malicious user breaks into talon.wolfpaw.net and destroys all DP files. Partial recovery from Orodreth's backups — player rent files and newer player data lost, along with significant portions of v3.0. darkpawns.com registered. Redesigned DP website unveiled.

**2004** — Game remains active and fully operational. Work continues on v3.0.
